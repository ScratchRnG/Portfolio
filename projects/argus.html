Host-Based Intrusion Detection System (HIDS) with AI-Assisted Analysis

Argus is a custom Host-Based Intrusion Detection System (HIDS) that integrates Suricata, real-time monitoring agents, and an AI-powered analysis dashboard to detect, visualize, and respond to malicious activity.

Overview

The system continuously monitors host and network activity using Suricata to detect suspicious and malicious traffic. Detected events are parsed, enriched, and securely forwarded to a custom-built web dashboard where analysts can view alerts, investigate trends, and receive automated guidance.

Key Features

Real-Time Threat Detection

Suricata analyzes traffic and generates alerts based on signature-based detection, which are streamed live to the dashboard.

Centralized Security Dashboard

A Flask-based dashboard provides:

  • Live alert feeds
  • Severity breakdowns
  • Alert trends over time
  • Source and destination IP visibility
  • Host CPU health and open port monitoring
  • Windows endpoint event ingestion and device health tracking

AI-Powered Security Assistant

An integrated AI chatbot (built with LangChain and Ollama) helps analysts interpret alerts, classifications, and system health data. The assistant is intentionally scoped to IDS-related context to provide concise, relevant explanations and reduce analyst fatigue.

Automated Email Alerting

When malicious traffic is detected, the system automatically sends email notifications to security personnel, ensuring timely awareness and rapid response.

Host Telemetry Collection

Lightweight background agents collect:

  • Open port information
  • CPU utilization
  • Windows event logs and device health

This data is correlated with network alerts to improve investigation context.

Secure Access & Authentication

The dashboard includes user authentication and session management to restrict access to authorized users.

Technologies Used

  • Detection & Monitoring: Suricata
  • Backend: Python, Flask, Flask-SocketIO
  • Frontend: HTML, CSS, JavaScript
  • AI Integration: LangChain, Ollama (LLM-based assistant)
  • Data Handling: JSON, SQLite
  • Alerting: SMTP Email Notifications
  • System Telemetry: psutil, custom monitoring agents
